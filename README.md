# create-backend-app

> cli tool for createing nodejs backend projects

## Usage

```
npx create-backend-app
```

## Supports

- [Express](https://expressjs.com/)
- [IBM AppID](https://www.ibm.com/cloud/app-id)
- [IBM Cloudant](https://www.ibm.com/cloud/cloudant)
